# Налоги гражданина РФ

Одностраничное веб-приложение для расчёта налогов гражданина РФ.

## Возможности

- Расчёт НДФЛ по прогрессивной шкале 2025 года
- Расчёт страховых взносов работодателя
- Учёт имущественных налогов
- Оценка косвенных налогов (НДС, акцизы)
- Визуализация распределения налогов
- Сохранение профиля в LocalStorage
- Шаринг через URL параметры

## Технологии

- React + TypeScript
- Vite
- Zustand (state management)
- Recharts (визуализация)
- Tailwind CSS + shadcn/ui
- Vitest (тестирование)

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка
npm run build

# Тесты
npm test
```

## Деплой

Приложение автоматически деплоится на GitHub Pages при push в ветку `main` через GitHub Actions.
